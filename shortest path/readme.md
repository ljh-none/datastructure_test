### MST
minimum (weight) spanning tree  
간선들의 가중치 합이 최소인 트리  
cycle 없이 모든 점을 연결시켜야 한다.(주의, tree에 간선 하나 추가하면 사이클됨)  

input = 1 connected component
그래프 상 점의 수 : n
spanning tree의 간선 수 : n-1
